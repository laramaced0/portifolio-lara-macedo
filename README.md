# Portfólio de Lara Macedo - UI/UX Designer

Portfólio web responsivo e moderno para apresentar projetos, experiências e competências como UI/UX Designer.

## 📁 Estrutura do Projeto

```
portifolio-dois/
├── index.html      # Página principal com toda a estrutura HTML
├── styles.css      # Estilos CSS responsivos
├── script.js       # JavaScript para interatividade
└── README.md       # Este arquivo
```

## 🎨 Características

- **Responsivo**: Totalmente adaptado para dispositivos móveis, tablets e desktops
- **Moderno**: Design limpo com gradientes, transições suaves e animações
- **Acessível**: Semântica HTML adequada e contraste de cores
- **Rápido**: Sem dependências externas, apenas HTML, CSS e JavaScript
- **Seções**:
  - Hero/Home
  - Sobre Mim
  - Processo Criativo
  - Trajetória (Experiência, Educação, Certificados)
  - Projetos/Estudos de Caso
  - Contato
  - Footer

## 🚀 Como Usar

1. **Abrir o arquivo**:
   - Abra `index.html` em um navegador web

2. **Personalizar conteúdo**:
   - Edite o arquivo `index.html` para adicionar seu conteúdo
   - Modifique `styles.css` para alterar cores e estilos
   - Atualize `script.js` para adicionar funcionalidades

3. **Cores principais**:
   - Primária: `#8B5CF6` (Roxo)
   - Texto Escuro: `#1F2937`
   - Texto Claro: `#6B7280`
   - Fundo Claro: `#F9FAFB`

## 📱 Responsividade

O portfólio é responsivo em 3 breakpoints:
- **Desktop**: 1200px+
- **Tablet**: 769px - 1199px
- **Mobile**: Até 768px

## ✨ Funcionalidades JavaScript

- Navegação suave (smooth scroll)
- Animações de entrada ao rolar a página
- Efeitos de hover nos botões
- Active link na navegação baseado na posição do scroll
- Intersection Observer para performance otimizada

## 📝 Dicas de Customização

### Mudar Cor Primária
No `styles.css`, localize `:root` e altere:
```css
--primary-color: #8B5CF6; /* Mude para sua cor desejada */
--primary-dark: #7C3AED;
```

### Adicionar Novas Seções
1. Adicione um novo `<section>` no HTML
2. Crie uma classe CSS correspondente
3. Adicione links de navegação no navbar

### Animar Mais Elementos
No `script.js`, adicione seletores ao `querySelectorAll()` para animar outros elementos ao rolar.
