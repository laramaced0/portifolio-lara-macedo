# Portfólio | Lara Macedo

Bem-vindo(a) ao repositório do meu portfólio profissional.

Este projeto foi desenvolvido para reunir minha trajetória como UI/UX Designer e Desenvolvedora Front-end, apresentando projetos, estudos de caso e soluções que refletem meu processo de trabalho, da concepção da ideia até sua implementação.

Mais do que uma vitrine de projetos, este portfólio foi pensado como uma experiência de navegação. Cada elemento foi projetado para comunicar de forma clara minhas competências, valorizando princípios de usabilidade, acessibilidade, consistência visual e boas práticas de desenvolvimento.

## Objetivos

- Apresentar minha evolução profissional em UI/UX e Front-end.
- Compartilhar estudos de caso e projetos autorais.
- Demonstrar meu processo de design e tomada de decisões.
- Evidenciar minhas habilidades em pesquisa, interface e desenvolvimento.
- Centralizar informações profissionais e formas de contato.

> 🚧 Este projeto está em constante evolução. Novas funcionalidades, melhorias e estudos de caso serão adicionados continuamente.
